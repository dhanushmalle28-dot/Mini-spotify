MINI SPOTIFY
 Mini Spotify is a lightweight music streaming application developed for Android using Kotlin and Android Studio.
The main objective of this project is to create a simple, user-friendly music player that allows users to browse, play, pause, and manage songs efficiently.

This application demonstrates concepts of Android development, Kotlin programming, UI/UX design, media playback using ExoPlayer, and Firebase/Local storage integration.
